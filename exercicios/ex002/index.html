Parágrafos e quebras de linha

Você pode escrever um parágrafo do jeito que você quiser

Para quebrar uma linha dentro de um parágrafo, basta usar a tag br, como nesse exemplo:
agora a linha está quebrada

Vamos adicionar alguns simbolos especiais: ® © ™ € £ ¥
No material em pdf tem outros símbolos

Vamos adicionar alguns emojis:
🖖
Todos os emojis iniciam com os simbolos "&#x" seguido de um código específico para cada emoji (pode conseguir no site emojipedia)
